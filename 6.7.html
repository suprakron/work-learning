การวาดภาพด้วยเครื่องมือต่าง ๆ ในโปรแกรม Adobe illustrator CS6

การวาดรูปทรงเรขาคณิตและรูปทรงสำเร็จรูป
การวาดรูปทรงเรขาคณิตด้วย Shape Tool เป็นการวาดรูปทรงเรขาคณิตต่าง ๆ ได้ โดยใช้เครื่องมือสำหรับวาดรูปทรงสำเร็จรูปภายในโปรแกรม ซึ่งมีให้เลือกใช้อยู่มากมาย ดังนี้

การวาดรูปสี่เหลี่ยมมุมฉากด้วย Rectangle Tool มีดังนี้
1 ) คลิกเลือกรูปแบบของเครื่องมือใน Rectangle Tool
2) ตัวชี้เมาส์จะเปลี่ยนป็นสัญลักษณ์หลังรากนั้นคลิกค้างไว้เพื่อคลิกวาดรูปบนอาร์ตบอร์ด

การใช้เมาส์ร่วมกับคีย์บอร์ดในการเลือก
- กด <Shift> จะทำให้ได้รูปสี่เหลี่ยมจัตุรัส ซึ่งมีด้านเท่ากันทั้ง 4 ด้าน
- กด *A1> จุดที่คลิกกลายเป็นจุดกึ่งกลาง และเป็นการวาดรูปสี่เหลี่ยมออกมาจากจุดกึ่งกลาง

การวาดรูปทรงอิสระด้วยยเครื่องมือต่างๆ
ในส่วนนี้กล่าวถึง Pen Too เป็นเครื่องมือที่ใช้วาดเส้นอิสระ โดยแต่ละส่วนของเส้นนั้น จะถูกเชื่อมต่อกันด้วยจุดยึด (Anchor) ซึ่งภายในกลุ่มของ Pen Tool จะมีเครื่องมือต่าง ๆ สำหรับปรับแต่งเส้นพาธให้เป็นรูปทรงต่าง ๆ ได้ตามที่ต้องการ ดังนี้

การวาดเส้นด้วยกลุ่มเครื่องมือ Pen Tool
เราใช้เครื่องมือ Pen Tool วาดได้ทั้งเส้นตรงและเส้นโค้งอิสระ โดยสามารถควบคุมเครื่องมือได้ง่ายและลงรายละเอียดได้อย่างเเม่นยำ เครื่องมือในกลุ่ม Pen Tool
1) วาดเส้นตรง เราสามารถใช้ Pen Tool สร้างเส้นตรง โดยกำหนดจุดยึดเพียง 2 จุด
คือ จุดเริ่มต้นและจุดปลาย โดยเลือกเครื่องมือ Pen Tol! ตัวชี้เมาส์จะเปลี่ยนเป็นรูปให้วาดจุดเริ่มต้น จากนั้นตัวชี้เมาส์จะเปลี่ยนไปเป็น ให้คลิกวาดปลาย เราจะได้เส้นตรงจากนั้น เราสามารถวาดพาธเพิ่มเติมให้กลายเป็นรูปทรงแบบเปิดหรือแบบปิด โดยให้จุดสิ้นสุดของพาธ เป็นจุดเดียวกับจุดเริมตันของพาธ ซึ่งตัวชี้เมาส์จะเปลี่ยนไปเพื่อบอกให้ทราบว่าเป็นเส้นพาธแบบปิด
2 ) วาดเส้นโค้ง เราสามารถใช้ Pen Too สร้างเส้นโค้ง โดยกำหนดแขนปรับความโค้งได้ตามต้องการดังตัวอย่างต่อไปนี้เส้นโค้งที่มีจุดเชื่อมต่อกันเป็นการเชื่อมต่อกันระหว่างเส้นโค้งกับเส้นโค้ง โดยเมื่อเส้นโค้งเหล่านั้นเชื่อมต่อกันจะเกิดเส้นโค้งที่มีลักษณะที่แตกต่างกัน

การวาดรูปทรงด้วยเครื่องมือดินสอ (Pencil Tool)
ในส่วนของเครื่องมือดินสอ (Pencl Tool) เป็นเครื่องมือสำหรับวาดเส้นให้เป็นรูปต่าง ๆได้อย่างอิสระตามการลากเมาส์ และเราสามารถทำการแก้ใขได้อย่างง่ายดาย โดยใช้เครื่องมือ Smooth Tool และ Path Eraser Too เป็นเส้นอิสระที่วาดด้วยเครื่องมือดินสอนี้ ประกอบไปด้วย จุดยึดที่จัดวางในแนวโค้งของเส้นโดยอัตโนมัติ ซึ่งขึ้นอยู่กับลักษณะเส้นที่เราวาด
การวาดเส้นอิสระด้วยเครื่องมือ (Pencil Tool) เราสามารถใช้ดินสอวาดเส้นพาธได้ทั้งแบบเปิดและแบบปิด มักจะนิยมใช้ดินสอในการสเกตซ์วาดโครงร่างของภาพขึ้นมาก่อน จากนั้นจึงมีการปรับแต่งเส้นขอบให้โค้งเรียบตามความต้องการอีกครั้ง โดยมีวิธีการดังนี้ ต่อไปนี้
1 ) คลิกเลือกเครื่องมือ Pencil Tool
2) จากนั้นสร้างรูปขึ้นมาโดยมีการวาดเชื่อมต่อปลายทั้งสองของเส้นพาธให้เป็นแบบปิด

การวาดภาพด้วย Paintbrush Tool
เป็นการวาดภาพเฉพาะเสันโครงร่างด้วย Paintbrush Tool ซึ่งเป็นพู่กันและสามารถเปลี่ยนเป็นหัวพู่กันได้ โดยจะใช้งานควบคู่ไปกับพาเนล Brushes เราสามารถเปิดใช้านพาเนล Bushes ได้โดยใช้คำสั่ง Window > Brushes
1 ) คลิกเลือกเครื่องมือ Paintbrush Tool
2) คลิกเลือกรูปแบบ หัวพู่กันที่เราต้องการใช้งาน
3) คลิกวาดภาพบนอาร์ตบอร์ด ตามตำแหน่งที่ต้องการ

การกำหนดลักษณะเส้น
รกำหนดเส้นลักษณะเส้นขอบภาพ (Stroke) เพื่อนำไปใช้สร้างภาพจากเครื่องมือ เช่น การกำหนดลักษณะของเส้นประและขนาดของเส้นหนาหรือบางตามที่เราต้องการ โดยเราสามารถกำหนดลักษณะของเส้นได้ ดังนี้
1) คลิกเลือกส่วนของภาพที่ต้องการกำหนดลักษณะเส้น
2) เลือกแถบคำสั่ง Window
3) เลือกคำสั่ง Stroke
4) กำหนดความหนาของเส้นขอบจาก Weight

การลบภาพด้วย Eraser Tool
เป็นการใช้เครื่องมือยางลบลบสิ่งที่เราวาดผิด ซึ่งมีหลักสำคัญ 2 ข้อคือ 1) ถ้ามีการเลือกภาพเครื่องมือลบเฉพาะภาพที่ถูกเลือก วัตถุที่อยู่ใกล้ ๆ ไม่ถูกลบแม้ลากเมาส์ไปโดนก็ตาม 2) ถ้าไม่มีการเลือกภาพใดเลย เครื่องมือนี้จะลบทุกวัตถุที่เมาส์ลากผ่านการลบภาพด้วยเครื่องมือยางลบทำได้ดังนี้
1 ) เลือกเครื่องมือ Eraser Tool
2) ใช้เมาส์คลิกลากบริเวณที่ต้องการลบ
