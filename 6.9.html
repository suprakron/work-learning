การทำงานกับสีและการระบายสีในโปรแกรม Adobe Illustrator CS6

การเลือกสีจากกล่องเครื่องมือ
เมื่อกำหนดขนาดชิ้นงานเรียบร้อยแล้ว ก่อนที่จะทำการลงสีพื้นหลังของขึ้นงาน
เราจะมาเริ่มต้นรู้จักกับการกำหนดสีโดยวิธีสะดวกที่สุด เพราะสามารถเลือกสีได้ทันทีจากกลองเครื่องมือ
1) สีพื้น (Fil) และสีเส้น (Stroke) เป็นการกำหนดสีพื้นและสีเส้นของวัตถุ
2) สีพื้นฐานของโปรแกรม (Defaut Color) เป็นการเปลี่ยนสีพื้นและสีเส้นขอบของวัตถุให้กลับมาเป็นสีพื้นฐาน คือ สีพื้นเป็นสีขาวและสีเส้นขอบเป็นสีดำ
3) สลับสีพื้นและสีเส้น (Swap Fill and Stroke) คลิกเมาส์ที่กับสีเส้นเพื่อสลับสีพื้นกับสีเส้นขอบของวัตถุ
การใส่สีให้กับวัตถุ (FiIl)
1) คลิกเลือกวัตถุด้วยเครื่องมือ Direct Selection Tool
2) เลือกสีที่ต้องการจากสีพื้น Fill
การไล่เฉดสี (Gradient Tool)
Gradient Tool เป็นคำสั่งที่ใช้ระบายสีแบบไล่โทนสีจากสีหนึ่งไปยังอีกสีหนึ่ง เราสามารถกำหนดสีแต่ละช่วงที่ใช้ และมุมของการไล่โทนสีได้อีกด้วย โดยจะใช้งานควบคู่กันระหว่าง พาเนล Gradient และ Gradient Tool และการกำหนดทิศทาง และการกระจายตัวของการไล่โทนสีตามต้องการมีดังนี้
1 ) คลิกเลือกวัตถุด้วย Selection Tool
2) คลิกเลือกไอคอน Gradient Tool
3) จะปรากฏพาเนล Gradient คลิกเลือกสีที่ต้องการและเปลี่ยนจุดศูนย์กลางและการกระจายตัวของสีใหม่
การระบายสีด้วยพู่กัน
การระบายสีด้วยพู่กัน เป็นการวาดเฉพาะเส้นโครงร่างด้วย Peintbrush Tool เป็นเหมือนพู่กัน และสามารถเปลี่ยนหัวพู่กันได้ โดยจะใช้งานควมคู่ไปกับพาเนล Bushes เราสามารถเปิดใช้งานพาเนล Bushes ได้โดยเลือกเมนูคำสั่ง Window > Brushes และมีรูปแบบการใช้งานให้เลือกใช้ได้อย่างมากมาย ดังนี้
1) คลิก Paintbrush Tool เพื่อระบายสีด้วยพู่กัน
2) คลิกเลือกรูปแบบหัวพู่กัน
3) คลิเมาส์วาดรูปแบบของหัวพู่กันลงตามความต้องการ
